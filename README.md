<h1>
TESJI Blog
</h1>
Repositorio Para El Curso Curso Profesional De Git Y Github
Se realizó un blog para un curso de Git y GitHub, en este
se abordaron los siguientes temas:
* Comandos básicos en Git.
* Instalación de Git en Windows, Linux o MacOS.
* Inicialización de repositorios locales.
* Creación de llaves SSH.
* Manejo de ramas.
* Merge de ramas.
* Pull de un repositorio remoto,
* Push hacia un repositorio remoto.
* Fork de repositorios de otras personas.
* Simulación de un deploy en un servidor.
